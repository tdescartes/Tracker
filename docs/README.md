# Tracker — Documentation Index

> Household Inventory & Personal Finance, unified.

This folder contains the complete product and engineering documentation for **Tracker** — a dual-platform (web + mobile) application that bridges pantry management with personal finance through AI-powered receipt scanning, smart budgeting, goal planning, and bank statement analysis.

## Using These Docs

**For the landing page**: Each feature doc (02–11) includes a `Landing Page Copy` section with headline, subheadline, three bullet points, and a call-to-action — ready to drop into any marketing page framework.

**For design improvement**: The [design system doc](./12-design-system.md) catalogs every UI pattern, interaction, loading state, and design token currently in use, with concrete improvement recommendations.

**For development**: The architecture, data model, and API reference docs serve as the engineering source of truth.
